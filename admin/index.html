GitHub Configuration

Required for Save & Publish

How it works: The CMS writes your content JSON and image files directly to GitHub via the API. GitHub Pages then serves the updated site automatically. No server or build step needed.

Token: GitHub → Settings → Developer settings → Personal access tokens (classic) → Generate → select repo scope only.

Token is stored in your browser only — never sent anywhere except GitHub's API.
